# GA project 02 refactored with React Hooks

## Overview

Refactored version of News app Project2 at General Assembly. Full Readme and previous version of code can be found [here](https://github.com/GRudinsky/sei-project-2). <br />
Live version can be found on [Heroku](https://newsonthemews.herokuapp.com/).

## Available Scripts

In the project directory, you can run:
```
$ yarn start
```
Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
